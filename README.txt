# Automation Task

This project automates the **Contact Us form filling** using **Selenium, Pytest, and Allure Reports**.  
Test data is read from **Excel**, and screenshots are captured after form submission.

---

⚙️ Setup Instructions

1. Clone the Repository

https://github.com/Agnidathank/Automationtaskprevise

2. Create and Activate Python Environment if needed

python -m venv venv

venv\Scripts\activate

3. Install Dependencies

pip install -r requirement.txt

4.Run tests

pytest --alluredir=reports
